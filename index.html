What This Application Tests:

  • Session persistence across multiple requests
  • Load balancer routing to the same server instance
  • JSESSIONID cookie management
  • Server identification and session tracking
  • Session invalidation and new session creation
  • Dynamic server switching during ongoing sessions

How to Test:

  1. Select a specific server or use auto mode
  2. Enable "Dynamic server switching" to test mid-session server changes
  3. Click "Start Session Test" to begin
  4. Observe server affinity through multiple page refreshes
  5. Use the "Switch Server" button to force a server change for the next request
  6. Watch the visit count increment while tracking server changes
  7. Check the Balancer Manager for load distribution
